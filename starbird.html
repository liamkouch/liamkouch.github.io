1.2 Cardinality and Countable Sets

Let $2\mathbb{N}$ denote the set of positive even integers. Then $2\mathbb{N}$ has the same cardinality as $\mathbb{N}$.
The set $\mathbb{Z}$ has the same cardinality as $\mathbb{N}$.
Every subset of $\mathbb{N}$ is either finite or has the same cardinality as $\mathbb{N}$.
A set is countable if and only if it is finite or has the same cardinality as $\mathbb{N}$. A set is uncountable if and only if it is not countable.
Every infinite set has a countably infinite subset.
A set is infinite if and only if there is an injection from the set into a proper subset of itself.
The union of two countable sets is countable.
The union of countably many countable sets is countable.
The set $\mathbb{Q}$ is countable.
The set of all finite subsets of a countable set is countable.
Suppose a submarine is moving in the plane along a straight line at a constant speed such that at each hour, the submarine is at a lattice point, that is, a point whose two coordinates are both integers. Suppose at each hour you can explode one depth charge at a lattice point that will hit the submarine if it is there. You do not know the submarine's direction, speed, or its current position. Prove that you can explode one depth charge each hour in such a way that you will be guaranteed to eventually hit the submarine.
Say the submarine is in the $\mathbb{R}^2$ plane. Since we don't know the direction, speed, or position of the submarine in the plane, the general form of the position of the submarine with respect to time is \[(x_0 + at, y_0 + bt)\] where $x_0$ is the initial x-coordinate, $a$ is the horizontal velocity, $y_0$ is the initial y-coordinate, $b$ is the vertical velocity, and $t$ is the time past the starting time. We have $x_0,a,y_0,b \in \mathbb{Z}$ and $t \in \mathbb{N}$. We know $\mathbb{Z}^4$ is a finite cartesian product of countably infinite sets and is therefore countable. Then there is the bijection $f \colon \mathbb{N} \to \mathbb{Z}^4$ defined as \[f(t) = (f_x(t),f_a(t), f_y(t), f_b(t))\] Since $f$ is a bijection it follows for any choice of $x_0, a, y_0,$ and $b$ there exists some $t \in \mathbb{N}$ such that $f(t) = (x_0,a,y_0,b)$. Since $\mathbb{Z}^4$ is countable it follows we can check every possible direction, speed, and original position of the submarine. Suppose we want to hit the submarine with initial horizontal position $x_0$, horizontal velocity $a$, initial vertical position $y_0$, and vertical velocity $b$. Then at time $t_0 = f^{-1}(x_0,a,y_0,b)$ we explode a depth charge at coordinate \[(x,y) = (f_x(t_0) + f_a(t_0)t_0, f_y(t_0) + f_b(t_0)t_0)\] It follows we can be guaranteed to eventually hit the submarine.

1.3 Uncountable Sets and Power Sets

The cardinality of the set of natural numbers is not the same as the cardinality of the set of real numbers. That is the set of real numbers is uncountable.
For any set $A$, the set of all subsets of $A$ is called the power set of $A$ and is denoted $2^A$.
Suppose $A = \{a,b,c\}$. Explicitly write out $2^A$, the power set of $A$.
If a set $A$ is finite, then the power set of $A$ has cardinality $2^{|A|}$, that is, $|2^A| = 2^{|A|}$.
For any set $A$, there is an injection from $A$ into $2^A$.
For a set $A$, let $P$ be the set of all functions from $A$ to the two-point set $\{0,1\}$. Then $|P| = |2^A|$.
There is a one-to-one correspondence between $2^\mathbb{N}$ and the set of all infinite sequences of $0$'s and $1$'s.
There is no surjection from a set $A$ onto $2^A$. Thus for any set $A$, the cardinality of $A$ is not the same as the cardinality of its power set.

2.2 Open Sets and the Definition of a Topological Space

Suppose $X$ is a set. Then $\tau$ is a topology on $X$ if and only if $\tau$ is a collection of subsets of $X$ such that
  1. $\varnothing \in \tau$
  2. $X \in \tau$
  3. if $U \in \tau$ and $V \in \tau$, then $U \cap V \in \tau$, and
  4. if $\{U_\alpha \}_{\alpha \in \lambda}$ is any collection of sets of $\tau$, then $\cup_{\alpha \in \lambda} U_\alpha \in \tau$.
A topological space is an ordered pair $(X,\tau)$, where $X$ is a set and $\tau$ is a topology on $X$.
A set $U \subset X$ is called an open set in $(X,\tau)$ if and only if $U \in \tau$.
Let $\{U_i\}_{i=1}^n$ be a finite collection of open sets in a topological $(X, \tau)$. Then $\cap_{i=1}^n U_i$ is open.
We prove by inducting on $n$. The base case for $n = 1$ is trivially true. For the inductive hypothesis assume $P(m)$ is true where $P(m)$ denotes the theorem statement for $n = m$. We show $P(m+1)$ is true. Now suppose we a collection of $m+1$ open sets, $\{U_i\}_{i=1}^{m+1}$. By the hypothesis it follows $\cap_{i=1}^m U_i$ is open. Since the intersection of two open sets is an open set we have $(\cap_{i=1}^m U_i) \cap U_{m+1} \in \tau$. Hence, $\cap_{i=1}^{m+1} U_i \in \tau$. This concludes the induction.
Why does your proof not prove the false statement that the infinite intersection of open sets is necessarily open?
The above proof only shows that a finite intersection is open. Counterexample: consider $\tau_\text{std}$ on $\mathbb{R}$. Then the intersection over the collection $\{I_i\}_{i=1}^n$ where $I_j = (-1/j,1/j)$ is just $\{0\}$. However, this is not open.
A set $U$ is open in a topological space $(X,\tau)$ if and only if for every point $x \in U$, there exists an open set $U_x$ such that $x \in U_x \subset U$.
For the forward direction take $U_x = U$. For the reverse direction consider $\cup_{x \in U} U_x$ and note it is open since it is a union of open sets. We show $U = \cup_{x \in U} U_x$. First we show $U \subset \cup_{x \in U} U_x$. Let $p \in U$. Then there exists a corresponding $U_p$. Then $p \in U_p \subset \cup_{x \in U} U_x$. Hence, $U \subset \cup_{x \in U} U_x$. Now we show $U \supset \cup_{x \in U} U_x$. Let $q \in \cup_{x \in U} U_x$. Then $q \in U_x$ for some $x \in U$. Moreover, $U_x \subset U$ for any $x$. Hence, $q \in U_x \subset U$. Therefore, $U \supset \cup_{x \in U} U_x$. It follows $U = \cup_{x \in U} U_x$. Furthermore, we know the right side is open so then $U$ must also be open.
In $\mathbb{R}^n$, the Euclidean distance between points $x = (x_1,\dots,x_n)$ and $y=(y_1,\dots,y_n)$ is defined as \[d(x,y) = (\sum_{i=1}^n (x_i-y_i)^2)^{1/2}\] The map $d\colon \mathbb{R}^n \times \mathbb{R}^n \to \mathbb{R}$ we defined is the standard metric on $\mathbb{R}^n$. Then an open ball of radius $\varepsilon > 0$ around point $p \in \mathbb{R}^n$ is the set \[B(p,\varepsilon) = \{x \mid d(p,x) < \varepsilon\}\] Moreover, the standard topology $\tau_{\text{std}}$ on $\mathbb{R}^n$ is defined as follows: a subset $U$ of $\mathbb{R}^n$ belongs to $\tau_{\text{std}}$ if and only if for each point $p$ of $U$ there is an $\varepsilon_p > 0$ such that $B(p,\varepsilon_p) \subset U$.
The $\tau_{\text{std}}$ is a topology on $\mathbb{R}^n$.
The empty set is open vacuously. For whole set $\mathbb{R}^n$ consider any point $p \in \mathbb{R}^n$. Then we can take the ball $B(p,1)$. Then $p \in B(p,1) \subset \mathbb{R}^n$. It follows the whole set is open. Now consider two open sets $U$ and $V$. Consider a point $p$ in their intersection $U \cap V$. Since $U$ and $V$ are open and $p \in U$ and $p \in V$ there exists $r_u,r_v > 0$ such that $B(p,r_u) \subset U$ and $B(p,r_v) \subset V$. Take $r = \min\{r_u,r_v\}$. Then $B(p,r) \subset U$ and $B(p,r)\subset V$. Hence, $B(p,r) \subset U \cap V$. Now consider a collection of open sets $\{U_\alpha\}_{\alpha \in \lambda}$. Take a point $p \in \cup_{\alpha \in \lambda} U_\alpha$. Then $p \in U_\beta$ for some $\beta \in \lambda$. Since $U_\beta$ is open we have $p \in B(p,r) \subset \cup_{\alpha \in \lambda} U_\alpha$ for some $r > 0$. Hence, the union over the collection $\{U_\alpha\}_{\alpha \in \lambda}$ is open. It follows $\tau_\text{std}$ is a topology.
Given a set $X$, let $\tau = 2^X$, the set of all subsets of $X$. This topology $\tau$ is called the discrete topology on $X$. We call the space $(X,2^X)$ a discrete topological space.
Given a set $X$, the collection $\tau = \{\varnothing, X\}$ is called the indiscrete topology on $X$. We call the space $(X, \{\varnothing,X\})$ an indiscrete topological space. In this space, there are only two open sets: the empty set and the entire set $X$.
Given a set $X$, the finite complement of cofinite topology on $X$ is described as follows: a subset $U$ of $X$ is open if and only if $X-U$ is finite or $U = \varnothing$.
Given a set $X$, the countable complement or cocountable topology on $X$ is described as follows: a subset $U$ of $X$ is open if and only if $X-U$ is countable or $U = \varnothing$.
Verify that the discrete, indiscrete, finite complement, and countable complement topologies are indeed topologies for any set $X$.
We first verify the discrete topology. Since every subset of $X$ is open, $\varnothing$ and $X$ itself are open. If $U$ and $V$ are open sets. Then their intersection is another subset of $X$. Hence, $U \cap V$ is open. Moreover, an arbitrary union of open sets is a subset of $X$ so then the union must also be open. It follows the discrete topology is in fact a topology.

Second we verify the indiscrete topology. By definition $\varnothing$ and $X$ are open. Since these are the only two sets, there is only one intersection to consider, $\varnothing \cap X = \varnothing$. Hence, open sets are closed under intersection. Consider an arbitrary collection of open sets $\{U_\alpha\}_{\alpha \in \lambda}$. If for all $\alpha \in \lambda$, $U_\alpha = \varnothing$, then the union over the collection is $\varnothing$. Otherwise if for some $\alpha \in \lambda$ we have $U_\alpha = X$, then the union of the collection is $X$. In either case, the union over the collection is open. It follows the indiscrete topology is indeed a topology.

Third, we verify the cofinite topoology. By definition the empty set is open. Since $X - X = \varnothing$ which is finite, it follows $X$ is open. Now consider two open sets $U$ and $V$. Then consider $X - (U \cap V)$. By De Morgan's law we have $(X - U) \cup (X - V)$. We know $X-U$ and $X-V$ are finite and the union of two finite sets is finite. Hence, $U \cap V$ is open. Now consider the arbitrary collection of open sets $\{U_\alpha\}_{\alpha \in \lambda}$. Consider $X-\cup_{\alpha \in \lambda} U_\alpha$. This equates to $\cap_{\alpha \in \lambda} (X - U_\alpha)$. We know $X - U_\beta$ for some $\beta \in \lambda$ is finite and $\cap_{\alpha \in \lambda} (X - U_\alpha) \subset X-\beta$. It follows $\cap_{\alpha \in \lambda} (X - U_\alpha)$ is finite. Hence, the arbitrary collection of open sets is in fact open. Therefore, the confinite topology is a topology.

Lastly, we verify the cocountable topology. By definition the empty set is again open. $X-X = \varnothing$ which is finite implying countable. Since the union of two countable sets is countable, it follows if $X-V$ and $X-U$ are countable, their intersection is as well. Consider the arbitrary collection of open sets $\{U_\alpha\}_{\alpha \in \lambda}$. Then consider $X - \cup_{\alpha \in \lambda}U_\alpha$. This is equivalent to $\cap_{\alpha \in \lambda} (X-U_\alpha)$. This is at most countable. Hence, the cocountable topology is a topology.

Let $(X,\tau)$ be a topological space, $A$ a subset of $X$, and $p$ a point in $X$. Then $p$ is a limit point of $A$ if and only if for each open set $U$ containing $p$, $(U-\{p\})\cap A \ne \varnothing$. Notice that $p$ may or may not belong to $A$.
Let $X = \mathbb{R}$ and $A = (1,2)$. Verify that $0$ is a limit point of $A$ in the indiscrete topology and the finite complement topology, but not in the standard topology nor the discrete topology on $\mathbb{R}$.
In the indiscrete topology, the only open set that contains $0$ is $\mathbb{R}$. Then $(\mathbb{R}-\{0\}) \cap (1,2) = (1,2)$. Hence, $0$ is a limit point of $A$ in the indiscrete topology. Now consider the cofinite topology. Let $U$ be an open neighborhood of $0$. Suppose $U \cap (1,2) = \varnothing$. Then $(1,2) \subset \mathbb{R}-U$. This contradicts the complement of $U$ being finite. Hence, $U \cap (1,2) \ne \varnothing$ and $0$ is a limit point of $(1,2)$ in the finite complement topology. For the standard topology and discrete topology take $(-1,1)$ as an open neighborhood. It follows $0$ is not a limit point of $(1,2)$ in these last two topologies.

  2. $\overline{A \cup B} = \overline{A} \cup \overline{B}$
We prove part (1) first. Let $p$ be a limit point of $A$. Then for any open neighborhood $U$ of $p$ we have $(U-\{p\}) \cap A \ne \varnothing$. Since $A \subset B$ it follows $(U-\{p\}) \cap B \ne \varnothing$. Then $p$ is a limit point of $B$. Thus $\overline{A} \subset \overline{B}$.

Now we prove part (2). First we show $\overline{A \cup B} \subset \overline{A} \cup \overline{B}$. Equivalently, we show $X-(\overline{A} \cup \overline{B}) \subset X-(\overline{A \cup B})$. Let $p \in X-(\overline{A} \cup \overline{B})$. Then there exists some neighborhoods $U$ and $V$ of $p$ such that $(U-\{p\}) \cap A = \varnothing$ and $(V-\{p\}) \cap B = \varnothing$. Then take $W = U \cap V$. Then $(W-\{p\}) \cap (A \cup B) = \varnothing$. It follows $p$ is not a limit point of $A \cup B$. Moreover, $p \notin A \cup B$. Thus, $p \in X - \overline{A \cup B}$. Now we show $\overline{A} \cup \overline{B} \subset \overline{A \cup B}$. Let $p \in \overline{A} \cup \overline{B}$. If $p \in \overline{A}$ then $p \in \overline{A \cup B}$ since $A \subset A \cup B$ and part (1) of this theorem. The same argument holds for when $p \in \overline{B}$. Thus $\overline{A} \cup \overline{B} \subset \overline{A \cup B}$.
We show the cantor set satisfies the criteria above. We construct the cantor set. Let $C_0 = \left[0,1\right]$. Then $C_{n-1}$ consists of $2^n$ closed intervals. For each interval, take out the middle third. Then this is defined to be $C_n$. For example, \[C_0 = \left[0,1\right] \] \[C_1 = \left[0,1/3\right] \cup \left[2/3, 1\right]\] \[C_2 = \left[0,1/9\right] \cup \left[2/9,3/9\right] \cup \left[6/9,7/9\right] \cup \left[8/9, 1\right]\] Generally we have \[C_n = I_n^1 \cup I_n^2 \cup I_n^3 \cup \cdots \cup I_n^{2^n}\] Where if $j < k$ then $x < y$ for all $x \in I_i^j, y \in I_i^k$ for all $j,k \in \{1,2,3,...,2^i\}$ and all $i \in \mathbb{N}\cup\{0\}$. Then the cantor set is defined to be $C = \cap_{i=1}^\infty C_i$. We now show $C$ is a subset of the interval $\left[0,1\right]$, closed, contains no non-empty open interval, and no point of $C$ is an isolated point. Since $\left[0,1\right] \supset C_0 \supset C_1 \supset \cdots$ it follows $C \subset \left[0,1\right]$. Moreover, $C$ is an infinite intersection of closed sets which is closed by Theorem 2.16. Now we show that $C$ contains no non-empty open interval. For sake of contradiction, suppose that $C$ contains a non-empty open interval $(s,t)$ where $s < t$ and $s,t \in \mathbb{R}$. Let $P(n)$ be the statement "Let $a,b\in\mathbb{R}$ and $a < b$. If $(a,b) \subset C_n$ then $|a-b| \leq 1/3^n$ for some non-negative integer $n$." We show that $p(n)$ is true for all $n \geq 0$ by induction. For the base case, consider $n=0$. Then suppose $(a,b) \subset C_0 = \left[0,1\right]$. Then we have $0 \leq a < b \leq 1$. It follows $|a-b| \leq 1 = 1/3^0$. It follows $P(0)$ is true. Now we move on to the inductive step. For the inductive hypothesis we assume $P(m)$ is true. Now assume $(a,b) \subset C_{m+1}$. Since $C_{m+1} \subset C_m$ it follows $(a,b) \subset C_m$. Then $C_m$ has $2^m$ disjoint closed intervals. \[C_m = I_m^1 \cup I_m^2 \cup I_m^3 \cup \cdots \cup I_m^{2^m}\] where $I_w^z$ is the zth closed interval for $C_w$. It follows $(a,b)$ is in exactly one of these intervals. Then let $(a,b) = I_m^k$ for some $k \in \{1,2,...,2^m\}$. Then $I_m^k=\left[c,d\right]$ where $c,d \in \mathbb{R}$. Since $(a,b)\subset\left[c,d\right]$ we have $c \leq a < b \leq d$. It follows $|a-b|\leq d-c$. For the sake of contradiction suppose $d-c>1/3^m$. Then there exists $p,q\in\mathbb{R}$ such that $(p,q) \subset C_m$ and $|p-q| > 1/3^m$; however, since $(p,q) \subset C_m$ we must have $|p-q| \leq 1/3^m$ by the inductive hypothesis. This is a contradiction. It follows $d-c\leq 1/3^m$. Since $(a,b)\subset \left[c,d\right] \subset C_m$ and by assumption $(a,b)\subset C_{m+1}$, it follows $(a,b)$ falls in the left or right third of $\left[c,d\right]$. Specifically, $(a,b) \subset \left[c,c+\frac{1}{3}(d-c)\right]$ or $(a,b) \subset \left[c+\frac{2}{3}(d-c),d\right]$. Then we have \[c \leq a < b \leq c+\frac{1}{3}(d-c)\] or \[c + \frac{2}{3}(d-c) \leq a < b \leq d\] In either case we have $|a-b| \leq (d-c)/3$. But $d-c \leq 1/3^m$ so then $|a-b| \leq 1/3^{m+1}$. It follows $P(m+1)$ is true. This concludes the induction. Hence, $P(n)$ is true for all $n \geq 0$. We assumed that $(s,t) \subset C$. Then $(s,t) \subset C_m$ for all $m \in \mathbb{N}\cup\{0\}$. Since we showed $P(n)$ is true for all $n \in \mathbb{N}\cup\{0\}$, it follows $|s-t| \leq 1/3^m$ for all $m$. Suppose $|s-t| > 0$. Then by the archimedean property there exists a sufficiently large $m$ such that $|s-t| > 1/3^m$. It follows $|s-t| = 0$. Moreover $(s,t) \in \left[0,1\right]$ so then $0 < s,t$ which implies $s=t$. But this is a contradiction since this implies $(s,t)$ is an empty open interval. Moreover, we assumed $s < t$. Hence, $C$ contains no non-empty open interval. Now we show $C$ contains no isolated points. Let $P^*(n)$ be the statement "let $x \in \mathbb{R}$. If $x \in C_n$ there exists $y \in \mathbb{R}$ such that $y\ne x$, $y \in C$, and $|x-y| \leq 1/3^n$ for some non-negative integer $n$." We show $P^*(n)$ is true for all non-negative integers. Suppose $x \in \mathbb{R}$ and $x \in C_i$ for some $i \in \mathbb{N}\cup\{0\}$. Then \[C_i = I_i^1 \cup I_i^2 \cup I_i^3 \cup \cdots \cup I_i^{2^i} \] Then $x \in I_i^j$ for some $j \in \{1,2,3,...,2^i\}$. Then $I_i^j = \left[c,d\right]$ where $c,d \in \mathbb{R}$ and $c < d$. Then $c \leq x \leq d$. If $x < d$ then let $y = d$ so then $(x,y) \subset C_i$. Otherwise if $x = d$ then let $y = c$ so then $(y,x) \subset C_i$. In either case, the hypothesis of $P(i)$ (note $P(i)$ not $P^*(i)$) is satisfied so then by the implication of $P(i)$ we have $|x-y| \leq 1/3^i$. Moreover $y \ne x$. Now we show $y \in C$. Notice $y =d$ or $y = c$ for $I_i^j$. If suffices to show that if $y$ is an endpoint of some closed interval $I_p^q$ (where $p \in \mathbb{N}\cup\{0\}$ and $q \in \{1,2,3,...,2^p\}$), then $y \in C$. Suppose $y$ is an endpoint of some interval in $C_k$ where $k\geq0$. Then \[ y \in I_k^\ell = \left[c_k^\ell, d_k^\ell \right] \] for some $\ell \in \{1,2,3,...,2^k\}$ and $c_k^\ell$ and $d_k^\ell$ are the corresponding endpoints. Since $y$ is an endpoint of $I_k^\ell$ we have either $y = c_k^\ell$ or $y = d_k^\ell$. If $y = c_k^\ell$ then \[ y \in I_{k+1}^{2\ell-1} = \left[y,d_{k+1}^{2\ell-1}\right]=\left[y,c_k^\ell+(d_k^\ell-c_k^\ell)/3\right] \] Then $y = c_{k+1}^{2\ell-1}$. If $y = d_k^\ell$ then \[ y \in I_{k+1}^{2\ell} = \left[c_{k+1}^{2\ell}, y\right] = \left[c_k^\ell+2(d_k^\ell-c_k^\ell)/3,y\right] \] Then $y = d_{k+1}^{2\ell}$. It follows $y$ is an endpoint of some interval in $C_{k+1}$. Hence, we have shown if $y$ is an endpoint of some interval in $C_k$ then $y$ is an endpoint of some interval in $C_{k+1}$. Recall that we showed that if $x \in C_i$ then there exists $x\ne y \in C_i$ such $|x-y|\leq 1/3^i$. Moreover, we take $y$ as an endpoint in some closed interval $I_i^j$. It follows $y$ is an endpoint of some interval in $C_i$. We have shown that if $y$ is an endpoint of some interval in $C_k$ then $y$ is an endpoint of some interval in $C_{k+1}$ for all $k\geq 0$. By induction, t follows $y$ is an endpoint of $C_m$ and therefore $y \in C_m$ for all $m \geq i$. Moreover, we have $C_0 \supset C_1 \supset \cdots \supset C_i$. It follows $y \in C_v$ (but not necessarily an endpoint of) for all $0 \leq v < i$. It follows $y \in C_w$ for all $w \in \mathbb{N} \cup \{0\}$. Thus $y \in C$. It follows $P^*(n)$ is true for all $n \in \mathbb{N}\cup\{0\}$. Using $P^*(n)$ we show there exists no isolated points in $C$. Let $x \in C$ and suppose we have the open set $U = (x-\epsilon, x+\epsilon)$ where $\epsilon\in\mathbb{R}$ and $\epsilon>0$. Then by the archimedean property there exists a sufficiently large $n \in \mathbb{N} \cup \{0\}$ such that $1/3^n < \epsilon$. Then let $m \in \mathbb{N} \cup \{0\}$ such that $1/3^m < \epsilon$. Since $x \in C$ it follows $x \in C_m$. Then by $P^*(m)$ it follows there exists a $y \in C$ such that $|x-y| \leq 1/3^m < \epsilon$. It follows $((x-\epsilon, x+\epsilon) - \{x\}) \cap C \ne \varnothing$ since $y \in ((x-\epsilon, x+\epsilon) - \{x\}) \cap C$ and $y \ne x$. Thus $x$ is a limit point and not an isolated point. We have shown the cantor set, denoted as $C$, is a non-empty subset of the closed unit interval $\left[0,1\right]$ that is closed, contains no non-empty open interval, and no isolated points.

2.4 Interior and Boundary

The interior of a set $A$ in a topological space $X$, denoted $A^\circ$, is defined as \[A^\circ = \bigcup_{U \subset A, U \in \tau} U,\] Points of $A^\circ$ are called interior points of $A$.
Let $A$ be a subset of a topological space $X$. Then $p$ is an interior point of $A$ if and only if there exists an open set $U$ with $p \in U \subset A$.
We prove the forward direction. Let $p$ be an interior point of $A$. Then $p \in \bigcup_{U\subset A, U \in \tau}U$. Then $p$ is in some open set $V$ where $V \subset A$. Altogether $p \in V \subset A$. For the reverse direction let $p \in W$ where $W \subset A$ and is open. Then $W \subset \bigcup_{U\subset A, U \in\tau}U$. It follows $p$ is an interior point.
Show that a set $U$ is open in a topological space $X$ if and only if every point of $U$ is an interior point of $U$.
We prove the forward direction. Let $U$ be an open set. Then for every point $p \in U$ there is an open set $U_p$ such that $p \in U_p \subset U$. By Theorem 2.26 it follows $p$ is an interior point. For the reverse direction suppose every point of $U$ is an interior point. Then for every point $p \in U$ there is an open neighborhood $U_p$ of $p$ such that $p \in U_p \subset U$. Then $U = \bigcup_{p \in U} U_p$. This is a union of open sets. Hence, $U$ is open.
The boundary of $A$, denoted $\partial A$, is defined to be $\overline{A} \cap \overline{X-A}$.
Let $A$ be a subset of a topological space $X$. Then $A^\circ$, $\partial A$, and $(X-A)^\circ$ form a partition on $X$.
We first show that these sets are disjoint. We show $A^\circ \cap \partial A = \varnothing$. Let $p \in A^\circ \cap \partial A$. Then $p \in \overline{A} \cap \overline{X-A}$. Then for every open set $U$ of $p$ we have $(U-\{p\}) \cap (X-A) \ne \varnothing$. Moreover, $p \in A$ since $p \in A^\circ$ so for every open set $U$ of $p$ we have in particular $U \cap (X-A) \ne \varnothing$. Since $p \in A^\circ$ there exists an open neighborhood $V$ such that $p \in V \subset A$. Then $V \cap (X-A) = \varnothing$. This is a contradiction to $p$ being a limit point of $X-A$. Hence, $A^\circ$ and $\partial A$ are disjoint. A similar argument show $\partial A$ and $(X-A)^\circ$ are disjoint. Lastly, it cannot be $p \in A^\circ$ and $p \in (X-A)^\circ$ because that would imply $p \in A$ and $p \in X-A$. Therefore, the sets $A^\circ, \partial A, (X-A)^\circ$ are disjoint.

Now we show every point in $X$ is in one of these sets. We show that if $p \notin (A^\circ \cup \partial A \cup (X-A)^\circ)$ then $p \notin X$. Let $p \notin A^\circ$ and $p \notin \partial A$. For the sake of contradiction, suppose $p \in A$. Then either $p$ is a limit point of $A$ or an isolated point of $A$. Consider the case where $p$ is a limit point of $A$. Then $p$ is in the closure of $A$. If $p$ is in the closure of $X-A$ then $p \in \partial A$ which is a contradiction. Thus, assume $p \notin \overline{X-A}$. Then there exists some open neighborhood $V$ of $p$ such that $V \cap (X-A) = \varnothing$ or equivalently $V \subset A$. Then we have $p \in V \subset A$ which means $p$ is an interior point, $p \in A^\circ$. This is a contradiction. Then $p$ cannot be a limit point of $A$ so assume $p$ is an isolated point of $A$. For the sake of contradiction suppose $p$ is not a limit point of $X-A$. Then there exists an open neighborhood $V$ of $p$ such that $(V-\{p\}) \cap X-A = \varnothing$. In particular, $V \cap X-A = \varnothing$ since $p \in A$. Then $V \subset A$. Then $p \in V \subset A$. Then $p \in A^\circ$. This is a contradiction so $p$ is a limit point of $X-A$. Hence, $p \in \overline{A} \cap \overline{X-A} = \partial A$. Once more, this is a contradiction. Therefore, $p$ cannot be in $A$ since assuming $p \in A$ leads to contradictions. Using the facts that $p \notin \partial A$ and $p \notin (X-A)^\circ$, we can use an analogous argument to show $p \notin X-A$. If $p \notin A$ and $p \notin X-A$, then $p \notin X$. Then there exists some open neighborhood $W$ of $p$ such that $(W-\{p\}) \cap A = \varnothing$. Then $(W-\{p\}) \subset X-A$.

2.5 Convergence of Sequences

A sequence in a topological space $X$ is a function from $\mathbb{N}$ to $X$. The image of $i$ under this function is a point of $X$ denoted as $x_i$ and we traditionally write the sequence by listing its images: $x_1,x_2,\dots$ or in shorted form: $(x_i)_{i \in \mathbb{N}}$.
A point $p \in X$ is a limit of the sequence $(x_i)_{i \in \mathbb{N}}$, or, equivalently, $(x_i)_{i \in \mathbb{N}}$ converges to $p$ (written $x_i \to p$), if and only if for every open set $U$ containing $p$, there is an $N \in \mathbb{N}$ such that for all $i > N$, the point $x_i$ is in $U$.
Let $A$ be a subset of the topological space $X$, and let $p$ be a point in $X$. If the set $\{x_i\}_{i \in \mathbb{N}} \subset A$ and $x_i \to p$, then $p$ is in the closure of $A$.
If $p \in A$ then $p$ is in the closure of $A$ so assume $p \notin A$. We show $p$ is a limit point of $A$. Take an open set $U$ of $p$. Since $x_i \to p$ there exists an $N \in \mathbb{N}$ such that $x_i \in U$ for all $i > N$. Moreover, we have $\{x_i\}_{i \in \mathbb{N}} \subset A$ and $p \notin A$ so then $(U-\{p\}) \cap A \ne \varnothing$. Hence, $p$ is a limit point of $A$ and in the closure of $A$.
In the standard topology on $\mathbb{R}^n$, if $p$ is a limit point of a set $A$, then there is a sequence of points in $A$ that converges to $p$.
Let $p$ be a limit point of the set $A$. Then we construct a sequence in $A$ that converges to $p$. For every $i \in \mathbb{N}$ take $x_i$ to be some point in $B(p,1/i) \cap A$. Moreover, $B(p,1/i) \cap A$ is non-empty for all $i$ since $p$ is a limit point of $A$. Now we show the sequence $(x_i)_{i\in\mathbb{N}}$ converges to $p$. Take an open set $U$ of $p$. Since $\mathbb{R}^n$ is endowed with the standard topology, for some $r > 0$ we have $B(p,r) \subset U$. Then for some $n \in \mathbb{N}$ we have $0 < 1/n < r$. It follows $x_i \in U$ for all $i \geq n$. Hence, our sequence converges to $p$.

3 Bases, Subspaces, Products: Creating New Spaces

3.1 Bases

Let $\tau$ be a topology on a set $X$, and let $\mathcal{B} \subset \tau$. Then $\mathcal{B}$ is a basis for the topology $\tau$ if and only if every open set in $\tau$ is the union of elements of $\mathcal{B}$. If $B \in \mathcal{B}$, we say $B$ is a basis element or basic open set. Note that $B$ is an element of the basis $\mathcal{B}$, but a subset of the space $X$.
Let $(X,\tau)$ be a top. space, and let $\mathcal{B}$ be a collection of subsets of $X$. Then $\mathcal{B}$ is a basis for $\tau$ if and only if
  1. $\mathcal{B} \subset \tau$, and
  2. for each set $U$ in $\tau$ and point $p$ in $U$ there is a set $V$ in $\mathcal{B}$ such that $p \in V \subset U$.
We prove the forward direction first. The first property is true by definition. Now let $U$ be an open set and $p \in U$. By definition of a basis we have $U = \cup_{V \in \mathcal{B}} V$. Then $p \in V$ for some $V \in \mathcal{B}$ and $V \subset U$. Now we prove the reverse direction. Let $U$ be an open set. By the second property, for every point $p$ in $U$ there exists a basic open set $V_p$ such that $p \in V_p$ and $V_p \in \mathcal{B}$. Take $W = \cup_{V_p \in \mathcal{B}}V_p$. We show $W = U$. First we show $W \subset U$. Let $p \in W$. then $p \in V_p \subset U$. Hence, $W \subset U$. Now we show $U \subset W$. Let $p \in U$. Then there is some $V_p$ such that $p \in V_p$. It follows $p \in \cup_{V_p \in \mathcal{B}}V_p = W$. Therefore, $U = W$.
Prove the following bases are in fact bases for the corresponding topology on $\mathbb{R}$.
  1. Let $\mathcal{B}_1 = \{(a,b) \subset \mathbb{R} \mid a,b \in \mathbb{Q}\}$. Show that $\mathcal{B}_1$ is a basis for the standard topology on $\mathbb{R}$.
  2. Let $\mathcal{B}_2 = \{(a,b) \cup (c,d) \subset \mathbb{R} \mid a < b < c < d \text{ are distinct irrational numbers}\}$. Show that $\mathcal{B}_2$ is also a basis for the standard topology on $\mathbb{R}$.
Suppose $X$ is a set and $\mathcal{B}$ is a collection of subsets of $X$. Then $\mathcal{B}$ is a basis for some topology on $X$ if and only if
  1. each point of $X$ is in some element of $\mathcal{B}$
  2. if $U$ and $V$ are sets in $\mathcal{B}$ and $p$ is a point in $U \cap V$, there is a set $W$ in $\mathcal{B}$ such that $p \in W \subset (U \cap V)$.
We prove the reverse direction first. Let $\mathcal{B}$ be a collection of subsets of $X$ such that properties (1) and (2) hold. We show that $\mathcal{B}$ is the basis for some topology on $X$. Specifically, we construct a topology, $\tau$ on $X$ and show $\mathcal{B}$ is a basis for $\tau$. Now define we define $\tau$: a set $U$ is in $\tau$ if and only if $U$ is a union of elements in $\mathcal{B}$ or is the empty set $\varnothing$. Now we verify $\tau$ is in fact a topology on $X$. By definition of $\tau$, $\varnothing \in \tau$. Now we show $X \in \tau$. By the first property of $\mathcal{B}$, for every point $p \in X$ there exists an element $B_p \in \mathcal{B}$ such that $p \in B_p$. Then $X = \cup_{p \in X}B_p$. This is a union of elements in $\mathcal{B}$ so $X$ is in fact open. Let $U$ and $V$ be open sets. We show $U \cap V$ is also open. Since $U$ and $V$ are open let $U = \cup_{C \in \mathcal{B}_1}C$ and $V = \cup_{D \in \mathcal{B}_2}D$ where $\mathcal{B}_1,\mathcal{B}_2 \subset \mathcal{B}$. Let $p \in U \cap V$. Then $p \in (\cup_{C \in \mathcal{B}_1}C) \bigcap (\cup_{D \in \mathcal{B}_2}D)$. Then $p \in C$ for some $C \in \mathcal{B}_1 \subset \mathcal{B}$ and $p \in D$ for some $D \in \mathcal{B}_2 \subset \mathcal{B}$. Then by the second property of $\mathcal{B}$, there a set $W_p \in \mathcal{B}$ such that $p \in W_p \subset C \cap D$. In particular, $p \in W_p \subset U \cap V$. Take $W = \cup_{p \in U \cap V} W_p$. Now we show $W = U \cap V$. If $p \in W$ then $p \in W_q$ for some $q \in U \cap V$. Moreover, $W_q \subset U \cap V$. Thus $p \in U \cap V$ and hence $W \subset U \cap V$. Let $p \in U \cap V$. Then there is an associated $W_p$ such that $p \in W_p \subset \cup_{q \in U\cap V}W_q = W$. Hence, $p \in W$ and $U \cap V \subset W$. Then $U\cap V = W$. Furthermore, $W$ is a union of elements in $\mathcal{B}$ so $W$ is open in $\tau$. Thus, $U \cap V$ must be in $\tau$. Suppose we have an arbitrary collection of sets $\{U_\alpha\}_{\alpha \in \lambda}$ such that $U_\alpha \in \tau$ for all $\alpha \in \lambda$. Then $\cup_{\alpha \in \lambda}U_\alpha$ is in $\tau$ since each $U_\alpha$ is a union of sets in $\mathcal{B}$. It follows $\tau$ is in fact a topology on $X$. Moreover, $\mathcal{B}$ is a basis for $\tau$ since by definition every set in $\tau$ is a union of elements in $\mathcal{B}$.

Now we prove the forward direction. We prove property (1) first. Let $\tau$ be an arbitrary topology on $X$. Then $X = \cup_{B \in \mathcal{B}'}B$ where $\mathcal{B}' \subset \mathcal{B}$. Let $p \in X$. Then $p \in B$ for some $B \in \mathcal{B}' \subset \mathcal{B}$. Hence, the first property holds. Now we prove the second property. Let $U,V \in \mathcal{B}$. Then $U,V \in \tau$. Then $U \cap V \in \tau$ and $U \cap V = \cup_{C \in \mathcal{B}''}C$ where $\mathcal{B}'' \subset \mathcal{B}$. Let $p \in U \cap V$. Then $p \in C$ for some $C \in \mathcal{B}'' \subset \mathcal{B}$. Therefore property (2) holds.
The lower limit topology on $\mathbb{R}$ is generated by a basis $\mathcal{B}$ where $B \in \mathcal{B}$ if and only if $B = \{x \in \mathbb{R} \mid a \leq x < b\}$ where $a,b \in \mathbb{R}$. This space is denoted as $\mathbb{R}_\text{LL}$.
Check that the basis proposed above for the lower limit topology is in fact a basis.
To check that $\mathcal{B}$ is a basis of some topology on $\mathbb{R}$ we implement Theorem 3.3. Let $x \in \mathbb{R}$. Then $x \in \left[x-1,x+1\right) \in \mathcal{B}$. Now suppose $U,V \in \mathcal{B}$ and $x \in U \cap V$. Let $U = \left[a,b\right)$ and $V = \left[c,d\right)$. Then let $\ell = \max\{a,c\}$ and $u = \min\{b,d\}$. It follows $x \in \left[\ell,u\right) \in \mathcal{B}$. It follows by Theorem 3.3 that $\mathcal{B}$ is indeed a basis for a topology on $\mathbb{R}$.
Every open set in $\mathbb{R}_\text{std}$ is an open set in $\mathbb{R}_\text{LL}$, but not vice versa.
Suppose $\tau$ and $\tau'$ are two topologies on the same underlying set $X$. If $\tau \subset \tau'$, then we say $\tau'$ is finer than $\tau$ and $\tau$ is coarser than $\tau'$. We say strictly coarser or strictly finer if $\tau \ne \tau'$.
Give an example of two topologies on $\mathbb{R}$ such that neither is finer than the other, that is, the two topologies are not comparable.
We construct a topological space called the Double Headed Snake. Let $\mathbb{R}_{+00}$ be the set consisting of the positive real numbers together with two points which are denoted as $0'$ and $0''$. Put a topology on it generated by a basis consisting of all intervals in $\mathbb{R}_+$ (positive reals) of the form $(a,b)$ or else of the form $(0,b) \cup \{0'\}$ or $(0,b) \cup \{0''\}$ for $a,b \in \mathbb{R}_+$.
Check that the collection of sets that are specified as a basis for the Double Headed Snake actually forms the basis for a topology.
We use Theorem 3.3. First denote $\mathcal{B}$ as proposed basis. Let $p \in \mathbb{R}_{+00}$. If $p \in \mathbb{R}_+$ then $p \in (0, p+1) \in \mathcal{B}$. If $p = 0'$ then $p \in (0,1)\cup\{0'\}$ and if $p = 0''$ then $p \in (0,1) \cup \{0''\}$. It follows every point in $\mathbb{R}_{+00}$ is contained in some element of $\mathcal{B}$. Now let $U,V \in \mathcal{B}$ and $p \in U \cap V$.

4.1 Hausdorff, Regular, and Normal Spaces

Let $(X,\tau)$ be a top. space.
  1. $X$ is a $T_1$-space if and only if for every pair $x,y$ of distinct points there are open sets $U,V$ such that $U$ contains $x$ but not $y$, and $V$ contains $y$ but not $x$.
  2. $X$ is Hausdorff, or a $T_2$-space, if and only if for every pair $x,y$ of disinct points there are disjoint open sets $U,V$ such that $x \in U$ and $y \in V$.
  3. $X$ is regular if and only if for every point $x \in X$ and closed set $A \subset X$ not containing $x$, there are disjoint open sets $U,V$ such that $x \in X$ and $A \subset V$. A $T_3$-space is any space that is both $T_1$ and regular.
  4. $X$ is normal if and only if for every pair of disjoint closed sets $A,B$ in $X$, there are disjoint open sets $U,V$ such that $A \subset U$ and $B \subset V$. A $T_4$-space is any space that is both $T_1$ and normal.
    5. The most important property of a $T_1$-space is that points are closed.
A space $(X,\tau)$ is $T_1$ if and only if every point in $X$ is a closed set.

4.4 The Normality Lemma

Then $p \in X - \cup_{i=1}^n \overline{U_i'}$ and $p \in V_n$. Then there is an open set $W_{n,p}$ such that $W_{n,p} \subset X - \cup_{i=1}^n \overline{U_i'}$ and $W_{n,p} \subset V_n$. Then note $V_j \subset X - \cup_{k=1}^j \overline{U_k'}$. Then $V_j' \cap U_i' = \varnothing$. This contradicts $p \in V_j' \cap U_i'$. Now suppose $j \lt i$. Then note $U_i' \subset X-\cup_{k=1}^{i-1}\overline{V_k'}$. Then $U_i' \cap V_j' = \varnothing$ which contradicts $p \in U_i' \cap V_j'$. Hence, $U \cap V = \varnothing$.

5.1 Separable Spaces, An Unfortunate Name

7.1 Continuous Functions

Let $X$ and $Y$ be topological spaces. A function or map $f \colon X \to Y$ is a continuous function if and only if for every open set $U$ in $Y$, $f^{-1}(U)$ is open in $X$.
Let $X$ and $Y$ be topological spaces, and let $f \colon X \to Y$ be a function. Then the following are equivalent:

  2. For every closed set $K$ in $Y$, the inverse image $f^{-1}(K)$ is closed in $X$.
  3. For every limit point $p$ of a set $A$ in $X$, the image $f(p)$ belongs to $\overline{f(A)}$.

We first show (1) implies (2). Let $K$ be closed in $Y$. Then $Y-K$ is open. Then $f^{-1}(Y-K)$ is open in $X$. Now we show $f^{-1}(Y-K) = X - f^{-1}(K)$. Let $p \in f^{-1}(Y-K)$. Then $f(p) \in Y-K$. Then $p \in X - f^{-1}(K)$. Since $X - f^{-1}(K)$ is open it follows $f^{-1}(K)$ is closed.

Now we show (2) implies (3). Let $p$ be a limit point of $A$. If $p \in A$ then $f(p) \in \overline{f(A)}$. Now suppose $p \notin A$. For the sake of contradiction suppose $f(p) \notin \overline{f(A)}$. Then $p \in X - f^{-1}(\overline{A})$. Note $X - f^{-1}(\overline{A})$ is open by part (2). Then there is some open neighborhood $U$ of $p$ such that $U \subset X - f^{-1}(\overline{A})$. Moreover, $p$ is a limit point of $A$ so then $(U - \{p\}) \cap A \ne \varnothing$. Let $a \in (U-\{p\}) \cap A$. Then $a \in f^{-1}(\overline{f(A)})$. But $a \in U \subset X - f^{-1}(\overline{f(A)})$. This is a contradiction. Hence, $f(p) \in \overline{f(A)}$.

Now we show (3) implies (4). Let $x \in X$ and $V$ be an open set that contains $f(x)$. For the sake of contradiction suppose for every open set $U$ containing $x$ we have $f(U) \cap (X-V) \ne \varnothing$. Then $(U-\{x\}) \cap f^{-1}(X-V) \ne \varnothing$. Then $x$ is a limit point of $f^{-1}(X-V)$. Then by part (3) we have $f(x)$ belonging to $\overline{X-V}$ and since $X-V$ is closed $\overline{X-V} = X-V$. Then $f(x) \in X-V$ and $f(x) \in V$. This is a contradiction. Hence, there is an open set $U$ containing $x$ such that $f(U) \subset V$.

Now we show part (4) implies part (1). Let $V$ be an open set in $Y$. Then we show $f^{-1}(V)$ is open. If $f(X) \cap V = \varnothing$ then $f^{-1}(V) = \varnothing$ which is open. Then consider the set $W = \cup_{p \in f^{-1}(V)} U_p$. This is a union of open sets which is open. Now we show $W = f^{-1}(V)$. Let $p \in W$. Then $p \in U_q$ for some $q \in f^{-1}(V)$. Moreover, $f(U_q) \subset V$. Hence, $p \in f^{-1}(V)$. It follows $W \subset f^{-1}(V)$. Now let $x \in f^{-1}(V)$. Then $x \in U_x \subset W$. It follows $W = f^{-1}(V)$. Since $W$ is open it follows $f^{-1}(V)$ is open.

A function $f\colon X \to Y$ is closed if and only if for every closed set $A$ in $X$, $f(A)$ is closed in $Y$. A function $f\colon X \to Y$ is open if and only if for every open set $U$ in $X$, $f(U)$ is open in $Y$.
If $X$ is normal and $f\colon X \to Y$ is continuous, surjective, and closed, then $Y$ is normal.
If $\{B_\alpha\}_{\alpha\in\lambda}$ is a basis for $X$ and $f\colon X \to Y$ is continuous, surjective, and open, then $\{f(B_\alpha)\}_{\alpha\in\lambda}$ is a basis for $Y$.
Let $V$ be an open set in $Y$. Then $f^{-1}(V)$ is open in $X$ since $f$ is continuous. Then $f^{-1}(V) = \cup_{\beta \in \mu}B_\beta$ where $\mu \subset \lambda$. It follows $V = \cup_{\beta \in \mu} f(B_\beta)$. Hence, $\{f(B_\alpha)\}_{\alpha\in\lambda}$ is a basis for $Y$.
If $X$ is 2nd countable and $f \colon X \to Y$ is continuous, surjective, and open, then $Y$ is 2nd countable.
Let $\{B_i\}_{i \in \mathbb{N}}$ be a countable basis for $X$. Then $\{f(B_i)\}_{i \in \mathbb{N}}$ is a countable basis for $Y$ by Theorem 7.22. It follows $Y$ is 2nd countable.
Let $X$ be compact, and let $Y$ be Hausdorff. Then any continuous function $f\colon X \to Y$ is closed.
Let $X$ be compact and 2nd countable, and let $Y$ be Hausdorff. If $f \colon X \to Y$ is continuous and surjective, then $Y$ is 2nd countable.

7.3 Homeomorphisms

A function $f\colon X \to Y$ is a homeomorphism if and only if $f$ is a continuous bijection and the inverse map $f^{-1} \colon Y \to X$ is also continuous.
Two topological spaces, $X$ and $Y$, are homeomorphic or topologically equivalent if and only if there exists a homeomorphism $f \colon X \to Y$.
