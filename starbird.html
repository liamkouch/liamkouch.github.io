1.2 Cardinality and Countable Sets

Let $2\mathbb{N}$ denote the set of positive even integers. Then $2\mathbb{N}$ has the same cardinality as $\mathbb{N}$.
The set $\mathbb{Z}$ has the same cardinality as $\mathbb{N}$.
Every subset of $\mathbb{N}$ is either finite or has the same cardinality as $\mathbb{N}$.
A set is countable if and only if it is finite or has the same cardinality as $\mathbb{N}$. A set is uncountable if and only if it is not countable.
Every infinite set has a countably infinite subset.
A set is infinite if and only if there is an injection from the set into a proper subset of itself.
The union of two countable sets is countable.
The union of countably many countable sets is countable.
The set $\mathbb{Q}$ is countable.
The set of all finite subsets of a countable set is countable.

1.3 Uncountable Sets and Power Sets

The cardinality of the set of natural numbers is not the same as the cardinality of the set of real numbers. That is the set of real numbers is uncountable.
For any set $A$, the set of all subsets of $A$ is called the power set of $A$ and is denoted $2^A$.
Suppose $A = \{a,b,c\}$. Explicitly write out $2^A$, the power set of $A$.
If a set $A$ is finite, then the power set of $A$ has cardinality $2^{|A|}$, that is, $|2^A| = 2^{|A|}$.
For any set $A$, there is an injection from $A$ into $2^A$.
For a set $A$, let $P$ be the set of all functions from $A$ to the two-point set $\{0,1\}$. Then $|P| = |2^A|$.
There is a one-to-one correspondence between $2^\mathbb{N}$ and the set of all infinite sequences of $0$'s and $1$'s.
There is no surjection from a set $A$ onto $2^A$. Thus for any set $A$, the cardinality of $A$ is not the same as the cardinality of its power set.

2.2 Open Sets and the Definition of a Topological Space

Suppose $X$ is a set. Then $\tau$ is a topology on $X$ if and only if $\tau$ is a collection of subsets of $X$ such that
  1. $\varnothing \in \tau$
  2. $X \in \tau$
  3. if $U \in \tau$ and $V \in \tau$, then $U \cap V \in \tau$, and
  4. if $\{U_\alpha \}_{\alpha \in \lambda}$ is any collection of sets of $\tau$, then $\cup_{\alpha \in \lambda} U_\alpha \in \tau$.
A topological space is an ordered pair $(X,\tau)$, where $X$ is a set and $\tau$ is a topology on $X$.
A set $U \subset X$ is called an open set in $(X,\tau)$ if and only if $U \in \tau$.
Let $\{U_i\}_{i=1}^n$ be a finite collection of open sets in a topological $(X, \tau)$. Then $\cap_{i=1}^n U_i$ is open.
We prove by inducting on $n$. The base case for $n = 1$ is trivially true. For the inductive hypothesis assume $P(m)$ is true where $P(m)$ denotes the theorem statement for $n = m$. We show $P(m+1)$ is true. Now suppose we a collection of $m+1$ open sets, $\{U_i\}_{i=1}^{m+1}$. By the hypothesis it follows $\cap_{i=1}^m U_i$ is open. Since the intersection of two open sets is an open set we have $(\cap_{i=1}^m U_i) \cap U_{m+1} \in \tau$. Hence, $\cap_{i=1}^{m+1} U_i \in \tau$. This concludes the induction.
Why does your proof not prove the false statement that the infinite intersection of open sets is necessarily open?
The above proof only shows that a finite intersection is open. Counterexample: consider $\tau_\text{std}$ on $\mathbb{R}$. Then the intersection over the collection $\{I_i\}_{i=1}^n$ where $I_j = (-1/j,1/j)$ is just $\{0\}$. However, this is not open.
A set $U$ is open in a topological space $(X,\tau)$ if and only if for every point $x \in U$, there exists an open set $U_x$ such that $x \in U_x \subset U$.
For the forward direction take $U_x = U$. For the reverse direction consider $\cup_{x \in U} U_x$ and note it is open since it is a union of open sets. We show $U = \cup_{x \in U} U_x$. First we show $U \subset \cup_{x \in U} U_x$. Let $p \in U$. Then there exists a corresponding $U_p$. Then $p \in U_p \subset \cup_{x \in U} U_x$. Hence, $U \subset \cup_{x \in U} U_x$. Now we show $U \supset \cup_{x \in U} U_x$. Let $q \in \cup_{x \in U} U_x$. Then $q \in U_x$ for some $x \in U$. Moreover, $U_x \subset U$ for any $x$. Hence, $q \in U_x \subset U$. Therefore, $U \supset \cup_{x \in U} U_x$. It follows $U = \cup_{x \in U} U_x$. Furthermore, we know the right side is open so then $U$ must also be open.
In $\mathbb{R}^n$, the Euclidean distance between points $x = (x_1,\dots,x_n)$ and $y=(y_1,\dots,y_n)$ is defined as \[d(x,y) = (\sum_{i=1}^n (x_i-y_i)^2)^{1/2}\] The map $d\colon \mathbb{R}^n \times \mathbb{R}^n \to \mathbb{R}$ we defined is the standard metric on $\mathbb{R}^n$. Then an open ball of radius $\varepsilon > 0$ around point $p \in \mathbb{R}^n$ is the set \[B(p,\varepsilon) = \{x \mid d(p,x) < \varepsilon\}\] Moreover, the standard topology $\tau_{\text{std}}$ on $\mathbb{R}^n$ is defined as follows: a subset $U$ of $\mathbb{R}^n$ belongs to $\tau_{\text{std}}$ if and only if for each point $p$ of $U$ there is an $\varepsilon_p > 0$ such that $B(p,\varepsilon_p) \subset U$.
The $\tau_{\text{std}}$ is a topology on $\mathbb{R}^n$.
The empty set is open vacuously. For whole set $\mathbb{R}^n$ consider any point $p \in \mathbb{R}^n$. Then we can take the ball $B(p,1)$. Then $p \in B(p,1) \subset \mathbb{R}^n$. It follows the whole set is open. Now consider two open sets $U$ and $V$. Consider a point $p$ in their intersection $U \cap V$. Since $U$ and $V$ are open and $p \in U$ and $p \in V$ there exists $r_u,r_v > 0$ such that $B(p,r_u) \subset U$ and $B(p,r_v) \subset V$. Take $r = \min\{r_u,r_v\}$. Then $B(p,r) \subset U$ and $B(p,r)\subset V$. Hence, $B(p,r) \subset U \cap V$. Now consider a collection of open sets $\{U_\alpha\}_{\alpha \in \lambda}$. Take a point $p \in \cup_{\alpha \in \lambda} U_\alpha$. Then $p \in U_\beta$ for some $\beta \in \lambda$. Since $U_\beta$ is open we have $p \in B(p,r) \subset \cup_{\alpha \in \lambda} U_\alpha$ for some $r > 0$. Hence, the union over the collection $\{U_\alpha\}_{\alpha \in \lambda}$ is open. It follows $\tau_\text{std}$ is a topology.
Given a set $X$, let $\tau = 2^X$, the set of all subsets of $X$. This topology $\tau$ is called the discrete topology on $X$. We call the space $(X,2^X)$ a discrete topological space.
Given a set $X$, the collection $\tau = \{\varnothing, X\}$ is called the indiscrete topology on $X$. We call the space $(X, \{\varnothing,X\})$ an indiscrete topological space. In this space, there are only two open sets: the empty set and the entire set $X$.
Given a set $X$, the finite complement of cofinite topology on $X$ is described as follows: a subset $U$ of $X$ is open if and only if $X-U$ is finite or $U = \varnothing$.
Given a set $X$, the countable complement or cocountable topology on $X$ is described as follows: a subset $U$ of $X$ is open if and only if $X-U$ is countable or $U = \varnothing$.
Verify that the discrete, indiscrete, finite complement, and countable complement topologies are indeed topologies for any set $X$.
We first verify the discrete topology. Since every subset of $X$ is open, $\varnothing$ and $X$ itself are open. If $U$ and $V$ are open sets. Then their intersection is another subset of $X$. Hence, $U \cap V$ is open. Moreover, an arbitrary union of open sets is a subset of $X$ so then the union must also be open. It follows the discrete topology is in fact a topology.

Second we verify the indiscrete topology. By definition $\varnothing$ and $X$ are open. Since these are the only two sets, there is only one intersection to consider, $\varnothing \cap X = \varnothing$. Hence, open sets are closed under intersection. Consider an arbitrary collection of open sets $\{U_\alpha\}_{\alpha \in \lambda}$. If for all $\alpha \in \lambda$, $U_\alpha = \varnothing$, then the union over the collection is $\varnothing$. Otherwise if for some $\alpha \in \lambda$ we have $U_\alpha = X$, then the union of the collection is $X$. In either case, the union over the collection is open. It follows the indiscrete topology is indeed a topology.

Third, we verify the cofinite topoology. By definition the empty set is open. Since $X - X = \varnothing$ which is finite, it follows $X$ is open. Now consider two open sets $U$ and $V$. Then consider $X - (U \cap V)$. By De Morgan's law we have $(X - U) \cup (X - V)$. We know $X-U$ and $X-V$ are finite and the union of two finite sets is finite. Hence, $U \cap V$ is open. Now consider the arbitrary collection of open sets $\{U_\alpha\}_{\alpha \in \lambda}$. Consider $X-\cup_{\alpha \in \lambda} U_\alpha$. This equates to $\cap_{\alpha \in \lambda} (X - U_\alpha)$. We know $X - U_\beta$ for some $\beta \in \lambda$ is finite and $\cap_{\alpha \in \lambda} (X - U_\alpha) \subset X-\beta$. It follows $\cap_{\alpha \in \lambda} (X - U_\alpha)$ is finite. Hence, the arbitrary collection of open sets is in fact open. Therefore, the confinite topology is a topology.

Lastly, we verify the cocountable topology. By definition the empty set is again open. $X-X = \varnothing$ which is finite implying countable. Since the union of two countable sets is countable, it follows if $X-V$ and $X-U$ are countable, their intersection is as well. Consider the arbitrary collection of open sets $\{U_\alpha\}_{\alpha \in \lambda}$. Then consider $X - \cup_{\alpha \in \lambda}U_\alpha$. This is equivalent to $\cap_{\alpha \in \lambda} (X-U_\alpha)$. This is at most countable. Hence, the cocountable topology is a topology.

2.3 Limit Points and Closed Sets

Let $(X,\tau)$ be a topological space, $A$ a subset of $X$, and $p$ a point in $X$. Then $p$ is a limit point of $A$ if and only if for each open set $U$ containing $p$, $(U-\{p\})\cap A \ne \varnothing$. Notice that $p$ may or may not belong to $A$.
Let $X = \mathbb{R}$ and $A = (1,2)$. Verify that $0$ is a limit point of $A$ in the indiscrete topology and the finite complement topology, but not in the standard topology nor the discrete topology on $\mathbb{R}$.
In the indiscrete topology, the only open set that contains $0$ is $\mathbb{R}$. Then $(\mathbb{R}-\{0\}) \cap (1,2) = (1,2)$. Hence, $0$ is a limit point of $A$ in the indiscrete topology. Now consider the cofinite topology. Let $U$ be an open neighborhood of $0$. Suppose $U \cap (1,2) = \varnothing$. Then $(1,2) \subset \mathbb{R}-U$. This contradicts the complement of $U$ being finite. Hence, $U \cap (1,2) \ne \varnothing$ and $0$ is a limit point of $(1,2)$ in the finite complement topology. For the standard topology and discrete topology take $(-1,1)$ as an open neighborhood. It follows $0$ is not a limit point of $(1,2)$ in these last two topologies.
Suppose $p \notin A$ in a topological space $(X,\tau)$. Then $p$ is not a limit point of $A$ if and only if there exists a neighborhood $U$ of $p$ such that $U \cap A = \varnothing$.
We prove the forward direction. Suppose $p$ is not a limit point of $A$. Then by definition of a limit point, for some open set $U$ containing $p$ $(U-\{p\}) \cap A = \varnothing$. Since $p \notin A$ it follows $U \cap A = \varnothing$. For the backward direction, suppose there exists a neighborhood $U$ of $p$ such that $U \cap A = \varnothing$. Then $(U - \{p\}) \cap A = \varnothing$. It follows $p$ is not a limit point of $A$.

For any topological space $(X,\tau)$ and $A \subset X$, the set $\overline{A}$ is closed. That is, for any set $A$ in a topological space, $\overline{\overline{A}} = \overline{A}$.
We show that $\overline{A}$ contains all its limit points. Let $p$ be a limit point of $\overline{A}$. If $p \in A$ then $p \in \overline{A}$. Thus, we assume $p \notin A$. Now to prove $p \in \overline{A}$ it is sufficient to show $p$ is a limit point of $A$ which is what we prove. Since $p$ is a limit point of $\overline{A}$, by definition we have $(U-\{p\}) \cap \overline{A} \ne \varnothing$ for any open neighborhood $U$ of $p$. Let $q$ be in this intersection. There are two cases. If $q \in A$ then we are done. Now consider the case where $q$ is a limit point of $A$. Since $U$ is also an open neighborhood of $q$ and $q$ is a limit point of $A$, it follows $(U-\{q\} \cap A \ne \varnothing)$. Note $p$ is not in this intersection since we assumed $p \notin A$. Then $(U-\{p\}) \cap A \ne \varnothing$. Hence, $p$ is a limit point of $A$. Then, $p \in \overline{A}$. Therefore, $\overline{A}$ is closed.
Let $(X,\tau)$ be a topological space. Then the set $A$ is closed if and only if $X-A$ is open.
We prove the forward direction first. Let $p \in X-A$. Since $A$ is closed, $p$ is not in $A$ and is not a limit point of $A$. Then there exists an open neighborhood $U$ of $p$ such that $U \cap A = \varnothing$. Then $U \subset X-A$. Altogether, we have $p \in U \subset X-A$. Hence, by Theorem 2.3 $X-A$ is open. Now we prove the reverse direction. Say $X-A$ is open. We show $A$ is closed. Let $q$ be a limit point of $A$. Then for every open neighborhood $V$ of $q$, $(V-\{q\}) \cap A \ne \varnothing$. For the sake of contradiction suppose $p \in X-A$. Since, $X-A$ is open, there exists some open neighborhood $W$ of $p$ such that $p \in W \subset X-A$. Then $(W-\{p\}) \cap A = \varnothing$. But this contradicts $p$ being a limit point of $A$. Hence, $p$ must be in $A$. Thus $A$ is closed.
Let $(X,\tau)$ be a topological space, and let $U$ be an open set and $A$ a closed subset of $X$. Then the set $U-A$ is open and the set $A-U$ is closed.
$U-A$ is equivalent to $U \cap (X-A)$. By Theorem 2.14 we know $X-A$ is open. It follows $U-A$ is open. Now we show $A-U$ is closed. Equivalently, we show $X-(A-U) = (X-A) \cup U$ is open. We know $X-A$ and $U$ are open so their union is open. Hence, $A-U$ is closed.
Let $(X,\tau)$ be a topological space. Then:
  1. $\varnothing$ is closed
  2. $X$ is closed
  3. The union of finitely many closed sets is closed
  4. Let $\{A_\alpha\}_{\alpha \in \lambda}$ be a collection of closed subsets in $(X,\tau)$. Then $\bigcap_{\alpha \in \lambda}A_\alpha$ is closed.
We utililze mainly Theorem 2.13. Since $X$ and $\varnothing$ are open we know their complements are closed, that is $\varnothing$ and $X$ are closed respectively. Suppose we have a finite union of closed sets $\cup_{i=1}^n U_i$. We show this is closed by showing the complement is open. The complement of this finite union is the finite intersection $\cap_{i=1}^n (X-U_i)$. This is a finite intersection of open sets. This is open. Hence, the finite union is closed. The complement of the intersection is the union $\bigcup_{\alpha \in \lambda} (X-A_\alpha)$. This is an arbitrary union of open sets which is open. Hence, the arbitrary intersection is closed.
For any set $A$ in a topological space $X$, the closure of $A$ equals the intersection of all closed sets containing $A$, that is, \[\overline{A} = \bigcap_{B \supset A, B \in C} B,\] where $C$ is the collection of all closed sets in $X$.
For convenience let $W = \bigcap_{B \supset A, B \in C} B$. Since $\overline{A} \supset A$ and $\overline{A} \in C$, it follows $W \subset \overline{A}$. Now we show $\overline{A} \subset W$. Take $p \in \overline{A}$. Suppose $p \in A$. If $B \supset A$ then $p \in B$. It follows $p \in W$. Now suppose $p \notin A$. Then $p$ is a limit point of $A$. Then for any open neighborhood $U$ of $p$ we have $(U-\{p\}) \cap A \ne \varnothing$. Since $A \subset B$ it follows $(U - \{p\}) \cap B \ne \varnothing$. Then $p$ is also a limit of $B$. Since $B$ is closed $p$ is in $B$. This holds for any $B$. Thus $p \in W$. Since $\overline{A} \subset W$ and $W \subset \overline{A}$ it follows $\overline{A} = W$.
Let $A$ and $B$ be subsets of a topological space $X$. Then:
  1. $A \subset B$ implies $\overline{A} \subset \overline{B}$
  2. $\overline{A \cup B} = \overline{A} \cup \overline{B}$
We prove part (1) first. Let $p$ be a limit point of $A$. Then for any open neighborhood $U$ of $p$ we have $(U-\{p\}) \cap A \ne \varnothing$. Since $A \subset B$ it follows $(U-\{p\}) \cap B \ne \varnothing$. Then $p$ is a limit point of $B$. Thus $\overline{A} \subset \overline{B}$.

Now we prove part (2). First we show $\overline{A \cup B} \subset \overline{A} \cup \overline{B}$. Equivalently, we show $X-(\overline{A} \cup \overline{B}) \subset X-(\overline{A \cup B})$. Let $p \in X-(\overline{A} \cup \overline{B})$. Then there exists some neighborhoods $U$ and $V$ of $p$ such that $(U-\{p\}) \cap A = \varnothing$ and $(V-\{p\}) \cap B = \varnothing$. Then take $W = U \cap V$. Then $(W-\{p\}) \cap (A \cup B) = \varnothing$. It follows $p$ is not a limit point of $A \cup B$. Moreover, $p \notin A \cup B$. Thus, $p \in X - \overline{A \cup B}$. Now we show $\overline{A} \cup \overline{B} \subset \overline{A \cup B}$. Let $p \in \overline{A} \cup \overline{B}$. If $p \in \overline{A}$ then $p \in \overline{A \cup B}$ since $A \subset A \cup B$ and part (1) of this theorem. The same argument holds for when $p \in \overline{B}$. Thus $\overline{A} \cup \overline{B} \subset \overline{A \cup B}$.
We show the cantor set satisfies the criteria above. We construct the cantor set. Let $C_0 = \left[0,1\right]$. Then $C_{n-1}$ consists of $2^n$ closed intervals. For each interval, take out the middle third. Then this is defined to be $C_n$. For example, \[C_0 = \left[0,1\right] \] \[C_1 = \left[0,1/3\right] \cup \left[2/3, 1\right]\] \[C_2 = \left[0,1/9\right] \cup \left[2/9,3/9\right] \cup \left[6/9,7/9\right] \cup \left[8/9, 1\right]\] Generally we have \[C_n = I_n^1 \cup I_n^2 \cup I_n^3 \cup \cdots \cup I_n^{2^n}\] Where if $j < k$ then $x < y$ for all $x \in I_i^j, y \in I_i^k$ for all $j,k \in \{1,2,3,...,2^i\}$ and all $i \in \mathbb{N}\cup\{0\}$. Then the cantor set is defined to be $C = \cap_{i=1}^\infty C_i$. We now show $C$ is a subset of the interval $\left[0,1\right]$, closed, contains no non-empty open interval, and no point of $C$ is an isolated point. Since $\left[0,1\right] \supset C_0 \supset C_1 \supset \cdots$ it follows $C \subset \left[0,1\right]$. Moreover, $C$ is an infinite intersection of closed sets which is closed by Theorem 2.16. Now we show that $C$ contains no non-empty open interval. For sake of contradiction, suppose that $C$ contains a non-empty open interval $(s,t)$ where $s < t$ and $s,t \in \mathbb{R}$. Let $P(n)$ be the statement "Let $a,b\in\mathbb{R}$ and $a < b$. If $(a,b) \subset C_n$ then $|a-b| \leq 1/3^n$ for some non-negative integer $n$." We show that $p(n)$ is true for all $n \geq 0$ by induction. For the base case, consider $n=0$. Then suppose $(a,b) \subset C_0 = \left[0,1\right]$. Then we have $0 \leq a < b \leq 1$. It follows $|a-b| \leq 1 = 1/3^0$. It follows $P(0)$ is true. Now we move on to the inductive step. For the inductive hypothesis we assume $P(m)$ is true. Now assume $(a,b) \subset C_{m+1}$. Since $C_{m+1} \subset C_m$ it follows $(a,b) \subset C_m$. Then $C_m$ has $2^m$ disjoint closed intervals. \[C_m = I_m^1 \cup I_m^2 \cup I_m^3 \cup \cdots \cup I_m^{2^m}\] where $I_w^z$ is the zth closed interval for $C_w$. It follows $(a,b)$ is in exactly one of these intervals. Then let $(a,b) = I_m^k$ for some $k \in \{1,2,...,2^m\}$. Then $I_m^k=\left[c,d\right]$ where $c,d \in \mathbb{R}$. Since $(a,b)\subset\left[c,d\right]$ we have $c \leq a < b \leq d$. It follows $|a-b|\leq d-c$. For the sake of contradiction suppose $d-c>1/3^m$. Then there exists $p,q\in\mathbb{R}$ such that $(p,q) \subset C_m$ and $|p-q| > 1/3^m$; however, since $(p,q) \subset C_m$ we must have $|p-q| \leq 1/3^m$ by the inductive hypothesis. This is a contradiction. It follows $d-c\leq 1/3^m$. Since $(a,b)\subset \left[c,d\right] \subset C_m$ and by assumption $(a,b)\subset C_{m+1}$, it follows $(a,b)$ falls in the left or right third of $\left[c,d\right]$. Specifically, $(a,b) \subset \left[c,c+\frac{1}{3}(d-c)\right]$ or $(a,b) \subset \left[c+\frac{2}{3}(d-c),d\right]$. Then we have \[c \leq a < b \leq c+\frac{1}{3}(d-c)\] or \[c + \frac{2}{3}(d-c) \leq a < b \leq d\] In either case we have $|a-b| \leq (d-c)/3$. But $d-c \leq 1/3^m$ so then $|a-b| \leq 1/3^{m+1}$. It follows $P(m+1)$ is true. This concludes the induction. Hence, $P(n)$ is true for all $n \geq 0$. We assumed that $(s,t) \subset C$. Then $(s,t) \subset C_m$ for all $m \in \mathbb{N}\cup\{0\}$. Since we showed $P(n)$ is true for all $n \in \mathbb{N}\cup\{0\}$, it follows $|s-t| \leq 1/3^m$ for all $m$. Suppose $|s-t| > 0$. Then by the archimedean property there exists a sufficiently large $m$ such that $|s-t| > 1/3^m$. It follows $|s-t| = 0$. Moreover $(s,t) \in \left[0,1\right]$ so then $0 < s,t$ which implies $s=t$. But this is a contradiction since this implies $(s,t)$ is an empty open interval. Moreover, we assumed $s < t$. Hence, $C$ contains no non-empty open interval. Now we show $C$ contains no isolated points. Let $P^*(n)$ be the statement "let $x \in \mathbb{R}$. If $x \in C_n$ there exists $y \in \mathbb{R}$ such that $y\ne x$, $y \in C$, and $|x-y| \leq 1/3^n$ for some non-negative integer $n$." We show $P^*(n)$ is true for all non-negative integers. Suppose $x \in \mathbb{R}$ and $x \in C_i$ for some $i \in \mathbb{N}\cup\{0\}$. Then \[C_i = I_i^1 \cup I_i^2 \cup I_i^3 \cup \cdots \cup I_i^{2^i} \] Then $x \in I_i^j$ for some $j \in \{1,2,3,...,2^i\}$. Then $I_i^j = \left[c,d\right]$ where $c,d \in \mathbb{R}$ and $c < d$. Then $c \leq x \leq d$. If $x < d$ then let $y = d$ so then $(x,y) \subset C_i$. Otherwise if $x = d$ then let $y = c$ so then $(y,x) \subset C_i$. In either case, the hypothesis of $P(i)$ (note $P(i)$ not $P^*(i)$) is satisfied so then by the implication of $P(i)$ we have $|x-y| \leq 1/3^i$. Moreover $y \ne x$. Now we show $y \in C$. Notice $y =d$ or $y = c$ for $I_i^j$. If suffices to show that if $y$ is an endpoint of some closed interval $I_p^q$ (where $p \in \mathbb{N}\cup\{0\}$ and $q \in \{1,2,3,...,2^p\}$), then $y \in C$. Suppose $y$ is an endpoint of some interval in $C_k$ where $k\geq0$. Then \[ y \in I_k^\ell = \left[c_k^\ell, d_k^\ell \right] \] for some $\ell \in \{1,2,3,...,2^k\}$ and $c_k^\ell$ and $d_k^\ell$ are the corresponding endpoints. Since $y$ is an endpoint of $I_k^\ell$ we have either $y = c_k^\ell$ or $y = d_k^\ell$. If $y = c_k^\ell$ then \[ y \in I_{k+1}^{2\ell-1} = \left[y,d_{k+1}^{2\ell-1}\right]=\left[y,c_k^\ell+(d_k^\ell-c_k^\ell)/3\right] \] Then $y = c_{k+1}^{2\ell-1}$. If $y = d_k^\ell$ then \[ y \in I_{k+1}^{2\ell} = \left[c_{k+1}^{2\ell}, y\right] = \left[c_k^\ell+2(d_k^\ell-c_k^\ell)/3,y\right] \] Then $y = d_{k+1}^{2\ell}$. It follows $y$ is an endpoint of some interval in $C_{k+1}$. Hence, we have shown if $y$ is an endpoint of some interval in $C_k$ then $y$ is an endpoint of some interval in $C_{k+1}$. Recall that we showed that if $x \in C_i$ then there exists $x\ne y \in C_i$ such $|x-y|\leq 1/3^i$. Moreover, we take $y$ as an endpoint in some closed interval $I_i^j$. It follows $y$ is an endpoint of some interval in $C_i$. We have shown that if $y$ is an endpoint of some interval in $C_k$ then $y$ is an endpoint of some interval in $C_{k+1}$ for all $k\geq 0$. By induction, t follows $y$ is an endpoint of $C_m$ and therefore $y \in C_m$ for all $m \geq i$. Moreover, we have $C_0 \supset C_1 \supset \cdots \supset C_i$. It follows $y \in C_v$ (but not necessarily an endpoint of) for all $0 \leq v < i$. It follows $y \in C_w$ for all $w \in \mathbb{N} \cup \{0\}$. Thus $y \in C$. It follows $P^*(n)$ is true for all $n \in \mathbb{N}\cup\{0\}$. Using $P^*(n)$ we show there exists no isolated points in $C$. Let $x \in C$ and suppose we have the open set $U = (x-\epsilon, x+\epsilon)$ where $\epsilon\in\mathbb{R}$ and $\epsilon>0$. Then by the archimedean property there exists a sufficiently large $n \in \mathbb{N} \cup \{0\}$ such that $1/3^n < \epsilon$. Then let $m \in \mathbb{N} \cup \{0\}$ such that $1/3^m < \epsilon$. Since $x \in C$ it follows $x \in C_m$. Then by $P^*(m)$ it follows there exists a $y \in C$ such that $|x-y| \leq 1/3^m < \epsilon$. It follows $((x-\epsilon, x+\epsilon) - \{x\}) \cap C \ne \varnothing$ since $y \in ((x-\epsilon, x+\epsilon) - \{x\}) \cap C$ and $y \ne x$. Thus $x$ is a limit point and not an isolated point. We have shown the cantor set, denoted as $C$, is a non-empty subset of the closed unit interval $\left[0,1\right]$ that is closed, contains no non-empty open interval, and no isolated points.

2.4 Interior and Boundary

The interior of a set $A$ in a topological space $X$, denoted $A^\circ$, is defined as \[A^\circ = \bigcup_{U \subset A, U \in \tau} U,\] Points of $A^\circ$ are called interior points of $A$.
Let $A$ be a subset of a topological space $X$. Then $p$ is an interior point of $A$ if and only if there exists an open set $U$ with $p \in U \subset A$.
Show that a set $U$ is open in a topological space $X$ if and only if every point of $U$ is an interior point of $U$.
The boundary of $A$, denoted $\partial A$, is defined to be $\overline{A} \cap \overline{X-A}$.
Let $A$ be a subset of a topological space $X$. Then $A^\circ$, $\partial A$, and $(X-A)^\circ$ form a partition on $X$.
