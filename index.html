Welcome. Unless you have seen a document made in LaTeX before, the layout of this site may be unfamiliar. LaTeX allows you to write neat documents, especially ones mathematical or scientific in nature. Here is more info. The source code for the styling of this site can be found on GitHub. This layout comes from a single CSS file. If you would like to try the layout add the following code in the figure below to the head tag of your project and you should be good to go. That's it. 

 <link rel="stylesheet" href="https://latex.now.sh/style.css">
